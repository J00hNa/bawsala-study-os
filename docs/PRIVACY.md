# Privacy, Retention, Export and Deletion

## البيانات المحلية

يحفظ التطبيق بيانات الدراسة في IndexedDB داخل المتصفح. مساحة `anonymous` منفصلة عن مساحة كل حساب `user:<id>`. تسجيل الخروج لا يجعل بيانات الحساب ظاهرة للملف المجهول، لكنه لا يمحو النسخة المحلية المخفية تلقائيًا. لمسحها، سجّل الدخول إلى الحساب ثم استخدم **Reset All Data**، أو امسح بيانات الموقع من إعدادات المتصفح بعد تصدير ما تحتاجه.

قد يحفظ التطبيق أيضًا معلومات صغيرة غير حساسة في Web Storage، مثل حالة المزامنة، معرّف جهاز عشوائي، وحالة المؤقت. جلسة Supabase نفسها تستخدم `sessionStorage` فتختفي عند إغلاق التبويب/جلسة المتصفح وفق سلوك المتصفح.

## البيانات السحابية

عند تفعيل الحساب، تُرفع حالة الدراسة إلى مشروع Supabase الذي يديره ناشر التطبيق. تُستخدم Row Level Security لعزل صفوف المستخدمين، لكن هذا ليس تشفيرًا طرفًا إلى طرف؛ مشغل المشروع وصاحب صلاحية الإدارة يقعان ضمن نموذج الثقة.

تحتوي السحابة على:

- حالة الدراسة الحالية.
- نسخ دورية محدودة للحالة السابقة.
- بيانات حساب Supabase اللازمة للمصادقة.
- سجل مزامنة محدود: request ID، النتيجة، revision، بصمة مختصرة لمعرّف جهاز عشوائي، والوقت. لا يسجل محتوى الملاحظات داخل سجل التدقيق.

## التصدير والاستعادة

**Export Data** ينشئ ملف JSON كاملًا. الملف غير مشفر؛ من يحصل عليه يستطيع قراءة محتواه. خزّنه في مكان خاص. الاستيراد يعرض مقارنة ويشترط تأكيدًا وينشئ نسخة محلية متحققًا منها قبل الاستبدال.

## الاحتفاظ

- النسخ المحلية: آخر 12 نسخة لكل namespace ما لم يحذفها المستخدم.
- النسخ السحابية: آخر 12 نسخة، وبحد أقصى نسخة آلية كل 15 دقيقة.
- سجل المزامنة: 90 يومًا لكل مستخدم.
- طلب حذف الحساب: فترة سماح 14 يومًا قبل الحذف النهائي.

## حذف الحساب

الحذف ليس فوريًا. يتطلب كلمة المرور وعبارة تأكيد، ثم يُجدول الطلب لمدة 14 يومًا. يمكن إلغاؤه بتسجيل الدخول قبل الموعد. وظيفة خادمية مجدولة هي الوحيدة التي تنفذ الحذف النهائي. النسخ المحلية في المتصفح لا تُمحى تلقائيًا عند حذف الحساب السحابي؛ امسح بيانات الموقع على الجهاز إذا أردت إزالة كل نسخة محلية.

## خدمات خارجية

عند تفعيلها، يستخدم التطبيق:

- Supabase للمصادقة وقاعدة البيانات وEdge Functions.
- Cloudflare Turnstile لمنع إساءة استخدام نماذج الحساب.
- مزود البريد الذي يختاره ناشر المشروع لرسائل التأكيد والاستعادة.

يجب على ناشر النسخة العامة استبدال هذا الملف بسياسة تحمل اسم الجهة، وسائل التواصل، الأساس القانوني، الدولة/القوانين المنطبقة، وفترات الاحتفاظ الفعلية. هذا الملف شرح تقني وليس استشارة قانونية.
